<h2 align="center">📖Sobre</h2>

<p align="center">Este projeto foi realizado para desenvolver melhor minhas habilidades em React, como também para uso pessoal, o projeto consiste em pode adicionar, editar e remover uma nova tarefa. Oprojeto é 100% responsivo.</p>

---

<h2 align="center">✔O site contém:</h2>

✅ Material UI<br>
✅ Animações<br>
✅ Editar item<br>
✅ Remover item<br>
✅ Responsividade<br>
### etc...
---

<h2>Instalando</h2>

```
npm install
```
ou
```
npm i
```

<h2>E logo em seguida:</h2>

```
npm start
```

<p>O projeto vai ser iniciado em <a target="_blank" href='http://localhost:3000'>http://localhost:3000</p>
